========================================================================
    MAKEFILE PROJECT : Docxygen Project Overview
========================================================================

This project will generate the documentation embedded in the comments
of the Network Alchemy implementation. Doxygen is used to generate the
document. Doxygen must be installed on the system for this project
to be able to generate the documentation. 

You can download doxygen from here:

http://sourceforge.net/projects/doxygen/files/latest/download
